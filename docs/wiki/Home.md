Seduction is part of the [Baudrillard Suite](https://github.com/bad-antics/baudrillard-suite) — a collection of Julia-based security tools named after Jean Baudrillard's works.

> **Ethical Use Only**: This tool is designed for defensive security awareness and training. Only use on systems and personnel you have authorization to test.
